# **App Name**: ExplainMate AI Tutor

## Core Features:

- Explanation Generation: Generates detailed, step-by-step explanations tailored to the student's profile using AI. Includes rough work derivations and real-world examples as a learning tool. Focus on clear and engaging content that breaks down complex topics for middle and high school students.
- Profile Personalization: Tailors AI responses based on the student's profile data (class, board, weak subjects) to adjust complexity and language, providing personalized educational content. The AI acts as a tool and references the students background to see if certain pieces of information will need further clarification.
- Tabbed Interface: Organizes AI responses into a tabbed interface (`Explanation`, `Rough Work`, `Real-World Examples`, `Fair Work`) for easy navigation and content review.
- Interactive Quizzes: Generates multiple-choice quizzes to reinforce understanding, including instant feedback with right/wrong prompts. Tailors question difficulty based on student performance, using AI to create dynamic learning assessments. It uses AI as a tool.
- Dynamic Sidebar: Provides a sidebar for navigation (New Explanation button, chat history) and settings access (About, Contact, etc.) and a persistent student profile for the app session.
- Welcome and Greeting: Personalized welcome message that greets returning students by name, creating a connection as the app launches
- Browser storage: Use the browser to store personal information. This does not require server connections, lowering complexity, cost, and barriers to deployment.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8216), mirroring the existing UI design which sets the tone as enthusiastic.
- Background color: Very light beige (#FFF9F0), for minimal distraction and maximized contrast.
- Foreground color: Dark brown (#61380A), used on text so content can easily be viewed on any device
- Accent color: Bright yellow (#FFC845) to highlight educational content.
- Headline font: 'Belleza' (serif) which communicates personality; body font: 'Alegreya' (serif).
- Uses brain circuit icon within a rounded square in orange.
- Employs sidebar-based layout with a clean, responsive design.
